# Aether: AI-Driven Data Analysis Platform

Aether is an advanced, AI-driven web application that enables users to have intuitive, conversational dialogues with their data. Upload CSV files, ask questions in plain English, and receive AI-powered responses with dynamically generated interactive charts. The innovative visual query loop allows users to circle points on charts and ask follow-up questions.

## Key Features
- **Conversational Data Analysis**: Natural language queries about your data
- **AI-Powered Visualizations**: Dynamic charts generated by Google Gemini
- **Visual Query Loop**: Select areas on charts and ask follow-up questions
- **Real-time Streaming Responses**: Get AI responses as they're generated
- **Multi-modal Interaction**: Combine text and visual inputs for deeper analysis

## Tech Stack
- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS, ECharts
- **Backend**: Go, Gorilla WebSocket, Redis
- **AI**: Google Gemini API
- **Deployment**: Docker & Docker Compose
